# Support

## Getting Help

If you need help with .github, here are some resources:

### Documentation

- Check the repository [README](https://github.com/zircote/.github#readme) for setup and usage instructions
- Browse the [wiki](https://github.com/zircote/.github/wiki) for detailed guides

### Community

- **Discussions**: Ask questions and share ideas in [GitHub Discussions](https://github.com/zircote/.github/discussions)
- **Issues**: Search [existing issues](https://github.com/zircote/.github/issues) for known problems and solutions

### Reporting Issues

- **Bugs**: [Open a bug report](https://github.com/zircote/.github/issues/new?template=bug_report.yml)
- **Feature requests**: [Open a feature request](https://github.com/zircote/.github/issues/new?template=feature_request.yml)
- **Security vulnerabilities**: See [SECURITY.md](SECURITY.md) for responsible disclosure instructions

## What to Include

When asking for help, please include:

1. What you were trying to do
2. What happened instead
3. Steps to reproduce the issue
4. Your environment details (OS, version, etc.)
5. Any relevant logs or error messages
